## 2. 시간 예산 ⏱️

> 상세 주차별 계획은 [[03-일정-로드맵]]

| 단계 | 시간 |
|------|:---:|
| 세팅 + 코어 (1주) | 8~10h |
| 룰 확장 + 테스트 (2주) | 8~10h |
| 데이터셋 + 평가 (3주) | 8~10h |
| 검증 + 문서화 (4주) | 6~8h |
| 마감 + 공개 (5주) | 6~8h |
| **핵심 합계** | **약 40~55h** |
| 확장(선택, 6주+) | +10~20h |

### 실행 시간 (도구 자체 성능)

| 모드 | 11MB APK 기준 | 용도 |
|------|:---:|------|
| `--fast` (매니페스트만) | ~1초 | 빠른 스캔 |
| 딥 모드 (코드까지) | ~38초 | 정밀 분석 |

→ 딥 모드가 느린 건 DEX 전체 문자열을 훑기 때문. 최적화는 [[08-확장-아이디어]].

## 4. 리스크 & 대비

| 리스크 | 대비책 |
|------|------|
| androguard 버전 API 변경 | requirements.txt로 버전 고정 (`>=4.1,<5`) |
| 딥 분석이 큰 APK에서 느림 | `--fast` 모드 제공, 문자열 상한(limit) 둠 |
| 악성 APK 다루다 감염 | **실행하지 않고 정적 분석만** → 안전 |
| 오탐(false positive) 많음 | 화이트리스트·심각도 조정 (3~4주차) |
